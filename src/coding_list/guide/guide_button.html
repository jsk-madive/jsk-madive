Button Guide

Basic Button

[Common]

- 기능에 따라 button태그 or a태그 사용
- button 태그 type=button 기본 설정, 용도에 따라 변경해주세요.
- 버튼 색상 [class=button] 요소 클래스 추가 : is-gray, is-blue, is-black
- 버튼 색상 전환 outline [class=button] 요소 클래스 추가 : is-inverted
- 버튼 크기 [class=button] 요소 클래스 추가 : is-small, is-medium, is-large
