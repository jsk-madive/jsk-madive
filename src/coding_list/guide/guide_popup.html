Popup Guide

[Common]

* [class=modal-open] id와 [class=modal] data-modal 동일값 선언하여 사용 *
- [class=modal-open] id 값 == [class=modal] data-modal 값
- 팝업 닫기 클래스
- 확인 : modal-confirm-btn
- 취소 : modal-del-btn
- 함수 호출
- Open + Close : modalEvent();
- Open : openModal();
- Close : closeModal();

Auto height Popup

고객님의 편리한 이용을 위해
아래 접근권한의 허용이 필요합니다.

위치(선택)
현재위치 자동수신
카메라 (선택)
리뷰 및 프로필 설정 시 촬영
사진 (선택)
