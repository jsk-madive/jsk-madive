spacing

[Common]

- margin, padding 여백 필요할 경우 클래스 선언하여 사용 가능
- 전체 여백(top, right, bottom, left) 조정
- margin 클래스 : m-0, m-1, m-2, m-3, m-4, m-5, m-6
- padding 클래스 : p-0, p-1, p-2, p-3, p-4, p-5, p-6
- 방향에 따른 여백 클래스 조정 가능
클래스 부여 숫자에 따른 여백 크기 설정 가능: *(0~6)
- [margin-top / padding-top] 클래스 : mt-* / pt-*
- [margin-right / padding-right] 클래스 : mr-* / pr-*
- [margin-bottom / padding-bottom] 클래스 : mb-* / pb-*
- [margin-left / padding-left] 클래스 : ml-* / pl-*
- [margin-left&right / padding-left&right] 클래스 : mx-* / px-*
- [margin-top&bottom / padding-top&bottom] 클래스 : my-* / py-*
상단 margin 설정
전체 padding 설정
좌우 margin 설정
<div class="mt-6">상단 margin 설정</div> <div class="p-6">전체 padding 설정</div> <div class="mx-4">좌우 margin 설정</div>
