Tab

[공통]

* [class=tabs] id와 [class=tab-contbox] data-tab 동일값 부여 필요 *
- [class=tabs] id 값 == [class=tab-contbox] data-tab 값
- tab 개수에 따른 [class=tab-list] 클래스 추가 : is-n (is-2, is-3, is-4 ...)
- tab title 사이즈 조정 가능 [class=tab-title]클래스 추가 : .is-1 ~ .is-6
- 함수실행 : tabAction();

Tab Basic

  • 음식배달
  • 전통시장
  • 쇼핑몰
<div class="tabs is-fullwidth"> <ul class="tab-list is-4" id="동일값"> <li class="on"> <a href="javascript:;" class="tab-title">음식배달</a> </li> <li> <a href="javascript:;" class="tab-title">전통시장</a> </li> <li> <a href="javascript:;" class="tab-title">쇼핑몰</a> </li> </ul> </div> <div id="contents" class="container"> <div class="tab-contbox" data-tab="동일값"> <div class="tab-cont"></div> <div class="tab-cont"></div> </div> </div> <!-- script --> <script> tabAction(); </script>
